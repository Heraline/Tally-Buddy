📲 Install on iPhone
⚠️

Must be opened in Safari — won't work in Chrome or other browsers

How to install — iPhone

01

Tap "Install on iPhone"

Tap the purple button above to download the profile

02

Allow the download

Tap Allow when Safari asks to download a configuration profile

03

Open Settings

Go to Settings → General → VPN & Device Management

04

Install the profile

Tap Family Wallet → tap Install → enter your passcode

05

Done! 🎉

How to install — Android

01

Open in Chrome

Tap the green button above to open Family Wallet in Chrome

02

Tap the menu

Tap the ⋮ three dots in the top right corner of Chrome

03

Add to Home Screen

Tap "Add to Home Screen" or "Install App"

04

Confirm install

Tap Add or Install on the popup that appears

05

Done! 🎉
