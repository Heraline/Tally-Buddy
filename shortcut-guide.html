Step 1 — Build the Shortcut (one-time setup)

1
Open Shortcuts app → tap +

Create a new shortcut. Name it "⚡ Family Wallet Auto Log". Tap the icon to give it a 💳 emoji.

2
Get latest screenshot

Add action: Get Latest Photos · Count: 1 · Enable Include Screenshots: On

3
Convert to Base64

Add action: Encode Media (search "encode")
Input: Photos · Encode as: Base64 · Line breaks: None

4
Call Gemini API

Add action: Get Contents of URL
URL: https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_KEY
Method: POST · Body type: JSON

// Paste this as the JSON body: { "contents": [{ "parts": [ { "inline_data": { "mime_type": "image/jpeg", "data": "[Encoded Media variable]" } }, { "text": "Extract the transaction from this receipt/payment screenshot. Return ONLY valid JSON, no markdown:\n{\"fw_shortcut\":true,\"amount\":0.00,\"desc\":\"merchant name max 30 chars\",\"cat\":\"food\",\"date\":\"YYYY-MM-DD\",\"cur\":\"MYR\",\"notes\":\"brief item summary max 40 chars\"}\ncat options: food transport shopping health entertainment utilities education other. Default cur to MYR." } ] }] }
💡 Get a free Gemini API key at aistudio.google.com → Create API Key
5
Extract the text response

Add: Get Dictionary from Input → input: Contents of URL
Add: Get Dictionary Value → Key: candidates
Add: Get Item from List → First Item
Add: Get Dictionary Value → Key: content
Add: Get Dictionary Value → Key: parts
Add: Get Item from List → First Item
Add: Get Dictionary Value → Key: text

6
Copy result to clipboard

Add action: Copy to Clipboard → Input: the text value from step 5

✅ Done! Family Wallet auto-detects this JSON when you open the app and pre-fills the form.
7
Optional: Auto-open Family Wallet

Add action: Open URLs → URL: https://heraline.github.io/Family-Wallet/
App opens and auto-fills immediately!

🔗 Share with family: Shortcut name → Share → Share Shortcut → copy iCloud link and send to everyone!

Step 2 — Daily use (2 taps)

1
Take a screenshot of any payment

GrabPay, Touch 'n Go, Maybank, DuitNow, food delivery, shopping receipts — anything showing an amount and merchant

2
Tap "⚡ Family Wallet Auto Log"

Run from your home screen widget, Back Tap, or Shortcuts app. Gemini reads the screenshot and copies the result.

3
Family Wallet opens pre-filled

Amount, merchant, category, date — all extracted automatically. Just review and tap Save!

⚡ Works with any payment screenshot — e-wallets, banking apps, food delivery, WhatsApp confirmations, photos of physical receipts

Make it even faster

★
Back Tap (fastest!)

Settings → Accessibility → Touch → Back Tap → Double Tap → assign the shortcut. Double-tap the back of your phone to log instantly!

★
Control Center (iOS 18+)

Settings → Control Center → add Shortcuts tile → pin the shortcut. One swipe + one tap from anywhere.

★
Action Button (iPhone 15 Pro+)

Settings → Action Button → Shortcuts → "⚡ Family Wallet Auto Log". Press the side button to log!

★
Home Screen Icon

Long-press shortcut → Share → Add to Home Screen. Tap its icon just like any other app.
