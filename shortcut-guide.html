Step 1 — Build the Shortcut (one time setup)

1
Open Shortcuts app → tap +

Create a new shortcut. Name it "Log Expense". Tap the icon to give it a custom emoji like 💳

2
Get latest screenshot

Add action: Get Latest Photos
Set Count: 1 · Enable: Include Screenshots: On

3
Convert to Base64

Add action: Encode Media (or Base64 Encode)
Input: Photos · Line breaks: None

4
Call Gemini API

Add action: Get Contents of URL
URL: https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_KEY
Method: POST · Body: JSON

// Paste this as the JSON body: { "contents": [{ "parts": [ { "inline_data": { "mime_type": "image/jpeg", "data": "[Base64 Encoded variable here]" } }, { "text": "Extract transaction from this receipt/payment screenshot. Return ONLY valid JSON, no markdown: {\"fw_shortcut\": true, \"amount\": 12.50, \"desc\": \"merchant name max 30 chars\", \"cat\": \"food\", \"date\": \"2026-03-29\", \"cur\": \"MYR\", \"notes\": \"brief item summary max 40 chars\"}. For cat use only: food, transport, shopping, health, entertainment, utilities, education, other. Default cur to MYR if not shown." } ] }] }
5
Extract the text response

Add action: Get Dictionary from Input → input: Contents of URL
Add: Get Dictionary Value → Key: candidates
Add: Get Item from List → First Item
Add: Get Dictionary Value → Key: content
Add: Get Dictionary Value → Key: parts
Add: Get Item from List → First Item
Add: Get Dictionary Value → Key: text

6
Copy result to clipboard

Add action: Copy to Clipboard
Input: the text value from step 5

✅ That's it! The JSON gets copied to your clipboard. Family Wallet reads it automatically when you open the app.
💡 Optionally add Open App → Family Wallet as the last action to jump straight to the PWA after copying

Step 2 — Daily use (2 taps)

1
Take a screenshot of any payment

GrabPay, Touch 'n Go, Maybank, DuitNow, food receipt — anything showing an amount and merchant

2
Tap "Log Expense" shortcut

From your home screen widget or Shortcuts app. Gemini reads the screenshot and copies the result.

3
Open Family Wallet PWA

The app automatically detects the clipboard data and opens the Add Transaction form pre-filled with amount, merchant, category and date.

4
Tap Save ✓

Review the details and save. Done!

⚡ Works with any payment screenshot — e-wallets, banking apps, food delivery, WhatsApp payment confirmations, physical receipt photos
